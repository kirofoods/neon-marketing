# Valorant Agent Voicelines

Place real Valorant agent voice MP3 files here.

## File Naming Convention
Each file must be named `{agentname}.mp3` (lowercase):

- `brimstone.mp3`
- `chamber.mp3`
- `cypher.mp3`
- `sova.mp3`
- `killjoy.mp3`
- `neon.mp3`
- `jett.mp3`
- `sage.mp3`
- `viper.mp3`
- `reyna.mp3`
- `phoenix.mp3`
- `astra.mp3`
- `fade.mp3`
- `gekko.mp3`
- `breach.mp3`
- `harbor.mp3`
- `deadlock.mp3`
- `kayo.mp3`
- `omen.mp3`
- `skye.mp3`
- `iso.mp3`
- `lobby.mp3`

## Where to Get Voice Lines
Extract from Valorant game files or download from community wikis.
The app will automatically play these MP3s when selecting agents.
If an MP3 is missing, it falls back to browser speech synthesis.
